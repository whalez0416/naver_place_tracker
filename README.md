# 🏆 네이버 & 구글 플레이스 순위 추적기 사용 설명서

본 프로그램은 **네이버 플레이스**와 **구글 지도(Local Pack)** 검색 결과에서 특정 업체(헤르지아)의 순위를 자동으로 추출하여 **구글 스프레드시트**에 기록하는 도구입니다.

---

## 🚀 1. 설치 및 실행 환경

### 필수 요구사항
- Python 3.8 이상
- Chrome 브라우저 설치됨

### 패키지 설치
최초 1회, 프로젝트 폴더에서 아래 명령어를 실행하여 필요한 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

---

## ⚙️ 2. 주요 설정 (config.py)

`config.py` 파일을 열어 아래 항목들을 관리할 수 있습니다.

| 설정 변수 | 설명 |
| :--- | :--- |
| `TARGET_PLACE_NAME` | 검색 결과에서 찾을 업체명 (**"헤르지아"**) |
| `TARGET_PLACE_ID` | 업체의 고유 ID (네이버용: `12780405`) |
| `KEYWORDS` | **네이버 + 구글** 양쪽 모두 순위를 체크할 한국어 키워드 목록 |
| `KEYWORDS_GOOGLE_ONLY` | **구글에서만** 체크할 키워드 (외국인 관광객용 영어 키워드 등) |
| `SPREADSHEET_ID` | 기록할 구글 스프레드시트의 ID |
| `HEADLESS_MODE` | `True` 설정 시 브라우저 창을 띄우지 않고 백그라운드에서 실행 |

---

## 📊 3. 구글 스프레드시트 연동

현재 모든 설정이 완료되어 있으며, 데이터는 다음 시트에 기록됩니다:
- **시트 주소**: [바로가기](https://docs.google.com/spreadsheets/d/1UoS5VprVDIy-fTfAq0As4F624XUQ7_dUPNNAEbUj99k/edit)
- **기록 항목**: 날짜, 시간, 매체(네이버/구글), 키워드, 순위(광고제외), 순위(전체), 광고여부, 업체명, **평점, 리뷰수**, 검색결과수

> [!IMPORTANT]
> 새로운 시트를 사용할 경우, 서비스 계정 이메일(`rank-tracker@naver-place-tracker.iam.gserviceaccount.com`)을 해당 시트에 **편집자** 권한으로 공유해야 합니다.

## 🏃 4. 실행 방법

터미널(또는 CMD)에서 아래 명령어를 입력하면 즉시 순위 추적을 시작합니다.
```bash
python tracker.py
```

### 실행 프로세스
1. Chrome 드라이버 자동 로드
2. `KEYWORDS` 목록 순회하며 네이버/구글 순위 추출 (스크롤/페이지네이션 자동 처리)
3. `KEYWORDS_GOOGLE_ONLY` 목록 순회하며 구글 전용 순위 추출
4. 결과를 구글 스프레드시트에 행 단위로 추가

---

## ⏰ 5. 자동 실행 (매일 3시간 간격)

현재 Windows 작업 스케줄러를 통해 **매일 3시간마다 백그라운드에서 자동 실행**되도록 설정되어 있습니다.

- **작업 이름**: `NaverRankTracker`
- **실행 파일**: `run_tracker.bat`
- **주기**: 매일 3시간마다
- **확인 방법**: Windows 검색창에서 **작업 스케줄러**를 실행 후 `NaverRankTracker` 작업이 있는지 확인.

수동으로 실행 파일(`run_tracker.bat` 또는 `python scheduler.py`)을 이용할 수도 있습니다.


## ⚠️ 주의사항 및 팁

- **순위권 밖**: 네이버는 상위 약 50~60개, 구글은 상위 100개 내에서 찾지 못하면 "순위권 밖"으로 기록됩니다.
- **차단 방지**: 검색 사이 사이 랜덤 대기 시간(3~7초)이 설정되어 있습니다. 너무 많은 키워드를 한꺼번에 조회하면 IP 차단 위험이 있으니 주의하세요.
- **구조 변경**: 네이버나 구글의 웹사이트 구조가 바뀌면 순위 추출이 안 될 수 있습니다. 이 경우 소스코드의 CSS 선택자(Selector) 업데이트가 필요합니다.
