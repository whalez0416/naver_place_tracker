# 🏆 네이버 & 구글 플레이스 순위 추적기 사용 설명서

본 프로그램은 **네이버 플레이스**와 **구글 지도(Local Pack)** 검색 결과에서 특정 업체(헤르지아)의 순위를 자동으로 추출하여 **구글 스프레드시트**에 기록하는 도구입니다.

**주요 기능**
- 네이버/구글 순위 자동 추출 + 구글시트 기록 (디자인 자동 적용)
- 직전 실행 대비 **순위 변동(▲/▼)** 자동 계산·표시
- **IP 차단 감지 + 자동 쿨다운**(차단 시 네이버 조회를 일정 시간 자동 중단)
- **로컬 CSV 백업**(시트 기록이 실패해도 데이터 보존)
- **텔레그램 알림**(순위 급변·IP 차단 시, 선택 설정)

### 📁 파일 구조
| 파일 | 역할 |
| :--- | :--- |
| `tracker.py` | 메인 실행 (크롤링 + 흐름 제어) |
| `config.py` | 모든 설정 (키워드, 시트, 알림, 쿨다운 등) |
| `anti_block.py` | 드라이버 생성 · 차단 감지 · 쿨다운 관리 |
| `sheet_io.py` | 구글시트 기록 · CSV 백업 · 순위 변동 · 시트 디자인 |
| `notifier.py` | 텔레그램 알림 |
| `scheduler.py` | 12시간 간격 자동 실행 (Windows 작업 스케줄러와 별개) |

## 🚀 1. 설치 및 실행 환경

### 필수 요구사항
- Python 3.8 이상
- Chrome 브라우저 설치됨

### 패키지 설치
최초 1회, 프로젝트 폴더에서 아래 명령어를 실행하여 필요한 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```

---

## ⚙️ 2. 주요 설정 (config.py)

`config.py` 파일을 열어 아래 항목들을 관리할 수 있습니다.

| 설정 변수 | 설명 |
| :--- | :--- |
| `TARGET_PLACE_NAME` | 검색 결과에서 찾을 업체명 (**"헤르지아"**) |
| `TARGET_PLACE_ID` | 업체의 고유 ID (네이버용: `12780405`) |
| `KEYWORDS` | **네이버 + 구글** 양쪽 모두 순위를 체크할 한국어 키워드 목록 |
| `KEYWORDS_GOOGLE_ONLY` | **구글에서만** 체크할 키워드 (외국인 관광객용 영어 키워드 등) |
| `SPREADSHEET_ID` | 기록할 구글 스프레드시트의 ID |
| `HEADLESS_MODE` | `True` 설정 시 브라우저 창을 띄우지 않고 백그라운드에서 실행 |

## 📊 3. 구글 스프레드시트 연동

현재 모든 설정이 완료되어 있으며, 데이터는 다음 시트에 기록됩니다:
- **시트 주소**: [바로가기](https://docs.google.com/spreadsheets/d/1UoS5VprVDIy-fTfAq0As4F624XUQ7_dUPNNAEbUj99k/edit)
- **기록 항목**: 날짜, 시간, 매체(네이버/구글), 키워드, 순위(광고제외), 순위(전체), 광고여부, 업체명, 평점, 리뷰수, 검색결과수, **변동**

### 🎨 시트 디자인 (클라이언트용 자동 적용)
새 시트이거나 헤더가 옛 버전이면 다음 디자인이 **자동으로** 적용됩니다.
- 헤더 행: 진한 파랑 배경 + 흰색 굵은 글씨 + 가운데 정렬 + **틀 고정**(스크롤해도 헤더 유지)
- 본문: 옅은 파랑/흰색 **줄무늬**로 가독성 향상
- **변동 컬럼**: 순위 상승(`▲`)은 초록, 하락(`▼`)은 빨강으로 자동 색상 표시

### 📈 순위 변동(변동 컬럼)
같은 키워드·매체의 **직전 기록 대비** 순위 변화를 자동 계산합니다.
- `▲3` = 순위 3계단 상승(좋아짐), `▼2` = 2계단 하락, `-` = 비교 대상 없음/순위권 밖

> [!IMPORTANT]
> 새로운 시트를 사용할 경우, 서비스 계정 이메일(`rank-tracker@naver-place-tracker.iam.gserviceaccount.com`)을 해당 시트에 **편집자** 권한으로 공유해야 합니다.

## 🏃 4. 실행 방법

터미널(또는 CMD)에서 아래 명령어를 입력하면 즉시 순위 추적을 시작합니다.
```bash
python tracker.py
```

### 실행 프로세스
1. Chrome 드라이버 자동 로드
2. `KEYWORDS` 목록 순회하며 네이버/구글 순위 추출 (스크롤/페이지네이션 자동 처리)
3. `KEYWORDS_GOOGLE_ONLY` 목록 순회하며 구글 전용 순위 추출
4. 결과를 구글 스프레드시트에 행 단위로 추가

---

## ⏰ 5. 자동 실행 (매일 2회 / 12시간 간격)

현재 Windows 작업 스케줄러를 통해 **12시간마다(하루 2회) 백그라운드에서 자동 실행**되도록 설정되어 있습니다.

- **작업 이름**: `NaverRankTracker`
- **실행 파일**: `run_tracker.bat`
- **주기**: 12시간마다 (하루 2회)
- **확인 방법**: Windows 검색창에서 **작업 스케줄러**를 실행 후 `NaverRankTracker` 작업이 있는지 확인.

> [!WARNING]
> 과거 3시간 간격(하루 8회)으로 실행하던 중 네이버가 **"과도한 접근 요청"으로 IP를 차단**한 사례가 있었습니다. 순위는 하루 1~2회면 충분하므로 **주기를 12시간보다 짧게 줄이지 마세요.**

수동으로 실행 파일(`run_tracker.bat` 또는 `python scheduler.py`)을 이용할 수도 있습니다.

---

## 🛡️ 6. 차단 회피 (자동 쿨다운)

네이버가 "과도한 접근"으로 차단하면, 계속 두드릴수록 차단이 길어집니다. 이를 막기 위해:

1. 차단을 감지하면 즉시 그 실행의 **네이버 조회를 중단**합니다. (구글은 계속)
2. `state.json`에 **쿨다운 종료 시각**을 기록하고, 그 시간(`config.NAVER_COOLDOWN_HOURS`, 기본 6시간)이 지날 때까지 **이후 실행에서도 네이버를 자동으로 건너뜁니다.**
3. 네이버 조회가 다시 정상화되면 쿨다운은 자동 해제됩니다.

> [!TIP]
> 탐지 회피를 더 강화하려면 `pip install -r requirements-optional.txt` 후 `config.py`에서 `USE_UNDETECTED_CHROMEDRIVER = True`로 변경하세요. (미설치 시 자동으로 일반 모드로 동작)

## 🔔 8. 텔레그램 알림 (선택)

순위가 크게 변하거나 IP가 차단되면 텔레그램으로 알림을 받을 수 있습니다.

1. `.env.example`을 복사해 `.env` 파일을 만듭니다.
2. [@BotFather](https://t.me/BotFather)로 봇을 만들어 **토큰**을, [@userinfobot](https://t.me/userinfobot)으로 **chat_id**를 확인해 `.env`에 채웁니다.
   ```
   TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
   TELEGRAM_CHAT_ID=123456789
   ```
3. `config.py`의 `NOTIFY_RANK_CHANGE_THRESHOLD`(기본 5)보다 순위 변동이 크면 알림이 옵니다.

값을 비워두면 알림 기능은 **자동으로 꺼지며** 나머지 기능에는 영향이 없습니다.

## ⚠️ 주의사항 및 팁

- **순위권 밖**: 네이버는 상위 약 50~60개, 구글은 상위 100개 내에서 찾지 못하면 "순위권 밖"으로 기록됩니다.
- **차단 방지**: 검색 사이 사이 랜덤 대기 시간(8~16초)이 설정되어 있습니다. 너무 많은 키워드를 한꺼번에/자주 조회하면 IP 차단 위험이 있으니 주의하세요.
- **IP 차단 감지**: 네이버/구글이 접근을 제한하면 로그에 `🚫 ... 차단/접근제한 감지됨` 이 남고, 네이버는 그 즉시 남은 조회를 **건너뛰어 차단이 더 길어지지 않게** 합니다. 차단된 키워드는 시트에 "순위권 밖"으로 잘못 기록되지 않습니다.
  - 이 메시지가 보이면 **몇 시간~하루 정도 실행을 멈추면** 대개 차단이 풀립니다. 공유기 재부팅으로 IP가 바뀌면 즉시 풀리기도 합니다.
- **구조 변경**: 네이버나 구글의 웹사이트 구조가 바뀌면 순위 추출이 안 될 수 있습니다. 이 경우 소스코드의 CSS 선택자(Selector) 업데이트가 필요합니다.
